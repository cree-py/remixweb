RemixBot

A multifunctional Discord bot in development that allows you to easily control your discord server.

Public Service Announcement

Previously, people have copied code from here and hosted their own instance without permission. We have trusted the public and made it a public repository again. Please do not host your own instance, instead, invite RemixBot here. If you wish to ask about or borrow any small snippets of code, please ask in our discord server before doing so. Thanks.

Contributing

If you need support with the bot or would like to contribute to RemixBot, join the discord server and ask any of the people with the RemixBot Devs role. Make sure to read the Contributor Covenant Code of Conduct and agree to it before becoming a contributor.

Features

  • Evalulate python code and more commands for developers!
  • Get info about a user, server, or role
  • Custom prefix
  • Moderator commands
  • Fun commands (8ball, Roll dice, etc.)
  • Mobile Game Stats (Clash Royale, Clash of Clans, Brawl Stars)
  • Utility (Embed, Urban Dictionary, Wikipedia)
  • Mod-logs

Official Release

This bot has finished the stages of it where there will be lots of bugs, and we have decided to release an early official version. It will still be actively worked on, but there will be less bugs. If you spot any, open an issue on this repo or use the suggest command.
